# Sales US Dashboard

A fully interactive **Sales Analytics Dashboard** project built as part of the *Tableau Ultimate Course* (Section 15: Tableau Project). This project visualizes and analyzes sales performance across the United States using real-world datasets.

---

## üìä Project Overview

The **Sales US Dashboard** demonstrates how to turn raw sales data into actionable insights through visualization and storytelling in Tableau. It showcases end-to-end data analysis ‚Äî from requirements gathering to dashboard publishing.

**Objective:**
To analyze sales trends, profits, and customer behavior across U.S. regions and product categories, enabling data-driven decision-making.

---

## üß© Project Phases

### 1. **Requirements Analysis**

* Define key business questions and KPIs (Sales, Profit, Quantity, Discount, etc.)
* Create a dashboard mockup and layout plan.
* Identify user needs and visualization goals.

### 2. **Data Preparation**

* Connect to the Superstore dataset (CSV or Excel format).
* Clean data and verify dimensions & measures.
* Create calculated fields for profit ratio, YoY growth, and profit contribution.

### 3. **Dashboard Design & Development**

* Create individual worksheets for:

  * Sales by Region
  * Profit by Category
  * Top 10 Customers
  * Sales vs. Profit Scatter Plot
  * Monthly Sales Trends
* Use containers to structure dashboard sections.
* Apply color consistency and UX design principles.

### 4. **Interactivity & Filters**

* Add filters for **Region**, **Category**, and **Year**.
* Use highlight and action filters for dynamic insights.
* Include KPIs and summary cards for instant readability.

### 5. **Testing & Validation**

* Check dashboard responsiveness and performance.
* Validate calculations and interactive behavior.
* Review with mock users or peers.

### 6. **Publishing**

* Publish dashboard to **Tableau Public**.
* Add title, description, and tags for discoverability.
